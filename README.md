🔗 URL-Based RAG Question Answering System

A Retrieval-Augmented Generation (RAG) application that answers user questions strictly based on the content of a provided YouTube link.
The project is Dockerized for reproducible and consistent execution.

✨ Features

🔗 URL-based knowledge ingestion

🧠 Context-grounded question answering (RAG)

🧩 Text chunking & semantic search

🔁 Supports follow-up queries using chat history

⚡ FastAPI backend

📦 Dockerized application


🛠 Tech Stack

FastAPI – Backend API

LangChain – RAG pipeline

HuggingFace Embeddings – Vector generation

FAISS – Vector storage & retrieval

Docker – Containerization

Uvicorn – ASGI server

🎯 Use Cases

Understanding YouTube video content

Building domain-specific assistants

Educational or research tools
